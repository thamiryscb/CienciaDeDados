# Datasets da Semana 3

Estes arquivos foram ajustados para a aula de limpeza, tratamento e padronização.
Eles contêm problemas intencionais, mas plausíveis, para apoiar o diagnóstico e a tomada de decisão pelos grupos.

## Tipos de problemas inseridos
- valores ausentes;
- categorias com grafias inconsistentes;
- espaços extras e diferenças de caixa;
- datas em formatos mistos e algumas datas inválidas;
- identificadores inconsistentes entre bases;
- duplicidades de registros e chaves;
- inconsistências entre campos relacionados.

## Arquivos
- `laboratorios_problematicos.xlsx`
- `uso_labs_problematicos.csv`
- `manutencao_problematico.json`
- `equipamentos_problematicos.csv`
