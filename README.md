📊 Ciência de Dados (DCA 3501)

Repositório dedicado ao armazenamento das atividades, conjuntos de dados (*datasets*) e anotações desenvolvidas durante a disciplina de **Ciência de Dados (DCA 3501)**.

---

## 🚀 Tecnologias e Ferramentas Utilizadas

Este projeto é desenvolvido com foco no ecossistema de análise de dados com Python:

* **Python** (Linguagem de programação principal)
* **Jupyter Notebook** (Formato dos arquivos `.ipynb`)
* **[Google Colaboratory](https://colab.research.google.com/)** (Ambiente em nuvem utilizado para escrever e executar os códigos)
* Bibliotecas comuns de Data Science (como Pandas, NumPy, Matplotlib, etc.)
